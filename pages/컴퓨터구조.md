- 주소지정방식
	- 즉치 주소지정방식 : ac <- nbr
	- 직접 주소지정방식 : ac <- m[adrs]
	- 간접 주소지정방식 : ac <- m[m[adrs]]
	- 상대 주소지정방식 : ac <- m[adrs + pc + 2]
	- 인덱스 주소지정방식 : ac <- m[adrs + r1]
	- 레지스터 주소지정방식 : ac <- r1
	- 레지스터 간접 주소지정방식 : ac <- m[r1]
- 제어단어의 각 필드
	- A,B : 출발 레지스터
	- D : 도착 레지스터
	- F : ALU 연산 하나선택
	- H : 시프터
