- 설치
	- brew로 vscode 설치
		- ```shell
		  brew install --cask visual-studio-code
		  ```
- 초기설정
	- setting.json
		- ```json
		  {
		    "editor.formatOnSave": true,
		    "editor.defaultFormatter": "esbenp.prettier-vscode",
		    "editor.fontSize": 15,
		    "editor.wordWrap": "on",
		    "window.zoomLevel": 1,
		    "git.autofetch": true,
		    "editor.fontFamily": "D2Coding",
		    "terminal.integrated.fontSize": 15
		  }
		  
		  ```
