		- 기타
			- snake case
			- ex) user_helper.ts
	- directory name
		- snake case
		- ex) date_handler
	- function name
		-
	- type
		- domain
			- pascal case
			- ```typescript
			  type User = {
			    ...
			  }
			  ```
		- state
			- 상태명 + State
			- ```typescript
			  type UserState = {
			    ...
			  }
			  ```
		- props
			- 컴포넌트명 + Props
			- ```typescript
			  type UserProfileProps = {
			    ...
			  }
			  ```
		- 기타

			- ```typescript
			  type Response = {
