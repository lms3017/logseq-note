- #resources
	- 내용
		- h, j, k, l : 좌, 하, 상, 우 커서 이동
		- i : 커서 위치에서 인서트
		- o : 커서 아래 위치에서 인서트
		- e : 단어 마지막 글자 기준으로 다음으로 이동
		- b : 단어 첫 글자 기준으로 이전으로 이동
		- H : 화면 맨 위로 이동
		- M : 화면 중간으로 이동
		- L : 화면 끝으로 이동
		- '>, < : 들여쓰기, 내어쓰기
		- [숫자]>, [숫자]< : 숫자만큼 들여쓰기, 내어쓰기
		- u : 실행취소
		- ctrl + r : 재실행
		- dd : 현재 줄 잘라내기
		- yy : 현재 줄 복사하기
		- [숫자]dd : 숫자만큼 잘라내기
		- [숫자]yy : 숫자만큼 복사하기
		- p : 붙여넣기
	- 출처
	- 태그
